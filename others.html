Prototype for Instant Microvascular Function Accessing

Developed a novel algorithm for Peripheral Artery Vasodilation function assessment by measuring microvascular flow signal using Skin Laser Doppler Method after artificial ischemia in the upper limb. Developed a user interface based on C# and finished a prototype for instant non-invasive assessment of microvascular function. Analyzed clinical data of 183 cases by statistical approach.

This poject won 2nd price in SJTU student innovation competition (Qian Xuesen Cup).
